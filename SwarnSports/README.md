# SwarnSports

Basic scaffold for aggregating cricket streaming links from Telegram.

## Structure

- `backend/`: Telegram fetching utilities and configuration.
- `frontend/`: Static UI shell for displaying aggregated links.

## Quick Start

1. Copy `.env.example` to `.env`.
2. Create a Python virtual environment.
3. Install dependencies from `backend/requirements.txt`.
4. Implement TODOs in backend and frontend files.
